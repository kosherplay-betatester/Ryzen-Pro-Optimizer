2. Test & Autotune Stability

Manual test — runs your current CO values through the selected stress test. If any core errors, the report tells you which one and suggests how much to dial back. You stay in the driver's seat.
Auto-Adjust (autotuning) — CoreCycler runs the stress test and automatically bumps each core's CO offset upward (less negative) whenever that core errors, then re-tests it. Walks each core to its individual stable edge. Set how far it can back off (Max value) and how aggressively it steps (Increment).
  • Starting values = your current per-core CO (read from SMU at test start)
  • Max value = the least-negative offset Auto-Adjust is allowed to reach. 0 means "give up at neutral"; 5 would allow positive offset (rarely useful, usually risky)
  • Increment = how many points to add per error. 1 is precise but slow; 3–5 finds the rough edge faster
  • Useful when you want to walk away and let it find each core's edge autonomously — common end-state: each core at its individual stable minimum
  • Watch the live status — it can take hours per core if errors come early in a long iteration

Default 1 cycle. 3+ cycles recommended for confident stability. Auto-Adjust typically benefits from longer runs (10+) so each core gets multiple chances.
